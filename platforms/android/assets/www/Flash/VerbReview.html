About

This page was developed in response to comments regarding the sibling Flash Cards page. It is intended to be a simple method of testing your knowledge of Spanish verbs. Just wave your mouse about and see what happens!

Usage

This site is most reliably viewed in Internet Explorer. It has some glitches in Mozilla and some bugs in Opera

To get information about a particular verb, simply hover your mouse over the entry that you are interested in. You should see a yellow box open up with the corresponding tense. Hover over the infinitive to get the English equivalent, or over the notes column to get information about any iregular tenses that are not shown or about other verbs that have the same oddities as this one.

Files

The data for the cards is held in XML files that contain a root element and a set of verb elements that each contain elements for the various tenses and parts of speech. As set up, only Spanish verbs are shown. Other languages can be simply accommodated - if anyone has the time to set up the data files I will change the code to allow a choice, as we do with the Flash Cards.

The elements contain CDATA text in UTF-8 format, which means that they can contain diacritical characters and simple HTML markup.

You could set up your own files using Notepad or a general purpose program (see below) but be sure that you save the file in UTF-8 format or the browser will have trouble rendering it.

If you would like more details on the file format, just get in touch.
